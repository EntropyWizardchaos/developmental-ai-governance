## What This Section Establishes

AI agents should not be born adult. The developmental framework defines:

**Childhood (externally supervised):** A mentor agent accepts or vetoes the learning agent's actions. Curated datasets. Limited permissions. High supervision. The agent learns THAT there are boundaries before learning WHERE they are.

**Adolescence (joint control):** The external mentor is gradually distilled into an internal critic (the IVS). The agent begins to self-monitor, with the mentor available for escalation. Permissions expand as coherence stabilizes.

**Adulthood (autonomous with conscience):** The internal critic modulates policy. Escalation triggers on coherence drops. Hard guardrails persist — they never vanish. The agent is autonomous but not unsupervised; it supervises itself through the internalized mentor.

**Death (productive mortality):** The agent's lifespan ends. Its ledger is sealed. Its wisdom is distilled into a sieve layer. The next generation inherits the filtered wisdom without inheriting the specific biases.

This is not anthropomorphism. It is recognition that the governance problems of autonomous systems are isomorphic to the governance problems of developing organisms. The solutions transfer because the structure transfers.
